常见问题

  • 1、 Orbiter Visa / Master卡是信用卡还是借记卡？

    Orbiter Visa / Master卡是一种预付卡。从广义上讲，预付卡与借记卡相同。不同之处在于借记卡与您的银行帐户相关联，但预付卡需要充值。在我们的产品中，您可以使用加密货币进行充值。

  • 2、如何申请 Orbiter Visa / Master卡？

    首先使用手机号或者邮箱注册Orbiter，下载APP之后按照提示步骤完成KYC，一般系统会花费2分钟时间来检验，通过之后即可在APP首页上点击申请，您将会立刻收到一张Orbiter Visa / Master卡,点击安全码后可以获得卡片详细信息。

  • 3、如何为我的Orbiter Visa / Master卡充值？

    您可以在 Orbiter应用程序中使用加密货币兑换成USD，比如使用USDT，然后将兑换好的USD充值到Orbiter Visa / Master卡中，即可开启全球消费。
